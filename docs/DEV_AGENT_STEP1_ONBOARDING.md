# DealDesk Dev Console: One-Step Start Sheet

## Single Step
Open Dev Console, paste your token, and send this exact message:

Show me PM2 status for dealdesk-backend-2 and the last 80 log lines.

## What Success Looks Like
1. You get process status and recent logs in chat.
2. Live Audit Log shows the action was executed.
3. No write approval modal appears because this is read-only.

## If It Fails
1. Check token value.
2. Check backend process is online.
3. Retry with a smaller request: "Show PM2 status only."
